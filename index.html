What is a pattern library?
Generally thought of as reusable user interface designs represented by html and css.
What is THIS pattern library?
A set of reusable html template files, coupled with styling by css and functionality from javascript. These html/css/js files are packaged with example data.
What makes them reusable?
The included html files are in one (or more) templating formats such as twig or mustache, which include {{curlyBraced}} placeholders for data. These files can be styled in your favorite design-in-browser tool after compiling with the included dummy data or imported into your cms system as raw templates.
How can I learn more and use it?
To see some example patterns, see the current pattern library on GitHub.
To play around with the code we're using to import these patterns into our projects, see our pattern importer.
